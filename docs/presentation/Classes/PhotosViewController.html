PhotosViewController

final class PhotosViewController : UIViewController, IndicatableController
extension PhotosViewController: ZoomAnimatorDelegate
extension PhotosViewController: FlexibleCollectionViewLayoutDelegate
extension PhotosViewController: InstantiableController
extension PhotosViewController: PhotoPageContainerViewControllerDelegate

Displays a list of photos. - It first shows a spinning wheel when retrieving the list of photos - If there is an error, it lets you try again - The number of photos displayed depends on the device and orientation - It only loads the thumbnal photos as the user scroll down to improve performance - It shows a nice animation when opening a photo detail or when closing it
